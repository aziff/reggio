This folder is to be incorporated into the Analysis directory of the Reggio project. Until the git repositories are aligned, this folder will house code by Jessica Koh and Anna Ziff.

Contents



Environment variables
Set up the following environment variables in order to run the code

global dir : env dir
This should point to one directory upstream from SURVEY_DATA_COLLECTION (on the athens server, this is klmReggio)

global datadir : env datadir
This should point to ${dir}/SURVEY_DATA_COLLECTION/data

global outdir : env outdir
This should point to the ${dir}/Analysis folder
